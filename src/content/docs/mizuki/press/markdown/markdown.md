Mizuki 主题使用标准 Markdown 语法编写文章内容。本文档涵盖所有基础语法元素及其使用方式。

## 块元素

### 段落与换行

段落之间使用**一个空行**分隔：

```markdown
这是第一段内容。

这是第二段内容。
```

在同一段落内换行，在行末添加**两个或更多空格**：

```markdown
这是一行内容。  
这是同一段落的下一行。
```

### 标题

使用 `#` 号标记标题，支持六级标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

:::tip
建议文章标题使用 `##` 二级标题，因为 `#` 一级标题已被文章的 `title` frontmatter 字段占用。
:::

### 引用块

使用 `>` 符号创建引用块：

```markdown
> 这是一段引用文字。
>
> 引用可以包含多个段落。
```

渲染效果：

> 这是一段引用文字。
>
> 引用可以包含多个段落。

引用可以嵌套：

```markdown
> 第一层引用
>
> > 第二层引用
```

### 列表

#### 无序列表

使用 `-`、`*` 或 `+` 创建无序列表：

```markdown
- 第一项
- 第二项
  - 嵌套项
  - 嵌套项
- 第三项
```

#### 有序列表

使用数字加 `.` 创建有序列表：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

#### 任务列表

使用 `- [ ]` 或 `- [x]` 创建任务列表：

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个待办事项
```

### 代码块

#### 行内代码

使用单个反引号包裹代码：

```markdown
使用 `console.log()` 输出内容。
```

#### 围栏代码块

使用三个反引号创建代码块，可指定语言以启用语法高亮：

````markdown
```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```
````

Mizuki 主题基于 [Expressive Code](https://expressive-code.com/) 渲染代码块，支持以下特性：

- 语法高亮（100+ 语言）
- 行号显示
- 行高亮标记
- 代码折叠
- 终端样式代码块

##### 行高亮

使用花括号标记需要高亮的行：

````markdown
```javascript {1, 3-5}
function hello() {
  console.log("Hello");
  console.log("World");
  console.log("!");
  return true;
}
```
````

##### 终端代码块

使用 `shell` 或 `bash` 语言标识创建终端样式：

````markdown
```shell
npm install
npm run dev
```
````

### 分隔线

使用三个或更多 `-`、`*` 或 `_` 创建水平分隔线：

```markdown
---
```

### 表格

使用竖线 `|` 和连字符 `-` 创建表格：

```markdown
| 列标题 1 | 列标题 2 | 列标题 3 |
|----------|----------|----------|
| 单元格   | 单元格   | 单元格   |
| 单元格   | 单元格   | 单元格   |
```

渲染效果：

| 列标题 1 | 列标题 2 | 列标题 3 |
|----------|----------|----------|
| 单元格   | 单元格   | 单元格   |
| 单元格   | 单元格   | 单元格   |

可以设置对齐方式：

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容   |   内容   |   内容 |
```

## 行内元素

### 链接

#### 普通链接

```markdown
[链接文本](https://example.com)
[带标题的链接](https://example.com "鼠标悬停标题")
```

#### 引用链接

```markdown
[链接文本][reference]

[reference]: https://example.com
```

### 强调

```markdown
*斜体文本* 或 _斜体文本_
**粗体文本** 或 __粗体文本__
***粗斜体*** 或 ___粗斜体___
```

渲染效果：

- *斜体文本*
- **粗体文本**
- ***粗斜体***

### 删除线

使用双波浪线标记删除文本：

```markdown
~~删除的文本~~
```

渲染效果：~~删除的文本~~

### 行内代码

```markdown
这是一段包含 `行内代码` 的文字。
```

### 图片

```markdown
![替代文字](图片路径 "可选标题")
```

图片支持以下路径格式：

1. **远程图片**：以 `http://` 或 `https://` 开头的完整 URL
2. **公共目录**：以 `/` 开头，引用 `public` 目录下的文件
3. **相对路径**：相对于当前 Markdown 文件的位置

## 数学公式

Mizuki 主题使用 KaTeX 渲染数学公式。

#### 行内公式

使用单个 `$` 包裹行内公式：

```markdown
爱因斯坦的质能方程 $E = mc^2$ 是物理学中最著名的公式之一。
```

#### 块级公式

使用双 `$$` 包裹块级公式：

```markdown
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

## 注意事项

1. Mizuki 主题基于 [Astro](https://astro.build/) 构建，Markdown 文件放在 `src/content/posts/` 目录下
2. 编码格式请使用 UTF-8
3. 建议在文件开头使用 frontmatter 元数据（参见 [文件管理](/press/file) 文档）
4. 图片推荐放在文章同级目录或 `public` 目录下
5. 更多增强功能请参见 [Markdown 增强功能](/press/customize) 文档
